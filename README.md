1- Project Overview

An interactive Excel dashboard developed to analyze Swiggy sales performance, customer ordering patterns, and restaurant trends. The dashboard combines Pivot Tables, Pivot Charts, Slicers, and KPI cards to transform raw sales data into meaningful business insights through an intuitive and user-friendly interface.

2- Project Objectives
- Monitor overall sales and order performance.
- Analyze customer purchasing trends over different time periods.
- Compare sales across states and cities.
- Evaluate food category performance.
- Enable interactive filtering for dynamic business analysis.

3- Dataset info :
The dashboard is built using a sample Swiggy sales dataset containing information such as:

- Order Details
- Restaurant Information
- Food Categories
- Customer Ratings
- Sales Amount
- State & City
- Order Date

4- Dashboard Features
- KPI Cards
- Total Sales
- Average Rating
- Average Order Value
- Total Ratings
- Total Orders
- Monthly Sales Trend Analysis
- Daily Sales Performance
- Food Type Sales Distribution (Veg vs Non-Veg)
- State-wise Sales Comparison
- Top 5 Cities by Sales
- Quarterly Sales, Ratings & Orders Summary
- Weekly Sales Trend
- Interactive Slicers
- Month
- Food Category 
- Restaurant Name

5- Key Business Insights
- Identifies the highest-performing states based on sales.
- Tracks monthly and weekly sales patterns.
- Compares the contribution of Veg and Non-Veg orders.
- Highlights top-performing cities by revenue.
- Measures customer engagement through ratings and order volume.
- Enables quick filtering to explore sales across different restaurants and categories.
